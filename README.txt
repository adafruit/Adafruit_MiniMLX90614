To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder Adafruit_MiniMLX90614. Check that the Adafruit_MLX90614 folder contains Adafruit_MiniMLX90614.cpp and Adafruit_MiniMLX90614.h

Place the Adafruit_MiniMLX90614 library folder your arduinosketchfolder/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
